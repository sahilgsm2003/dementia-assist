## ðŸŽ¯ CORE PRODUCT VISION

**Moments is a compassionate memory care companion that helps people with dementia and their caregivers navigate daily life with dignity, safety, and connection.**

### Key Principles:
1. **Simplicity First** - Every feature must be immediately understandable
2. **Safety Critical** - Emergency information always accessible
3. **Gentle Guidance** - Support, not replacement of human connection
4. **Multi-Stakeholder** - Serves both the person and their caregivers
5. **Progressive Disclosure** - Simple for basic use, powerful when needed

## ðŸŽ¨ UI/UX DESIGN PRINCIPLES

### 1. **Clarity Over Cleverness**
- Every screen should be immediately understandable
- No jargon or vague terms
- Clear labels: "My People" not "Memory Vault"

### 2. **Large & Accessible**
- Minimum 18px font size
- Large touch targets (min 44x44px)
- High contrast
- Simple language

### 3. **Progressive Disclosure**
- Simple view by default
- More details available if needed
- Don't overwhelm

### 4. **Visual Hierarchy**
- Most important information largest
- Use whitespace generously
- Clear visual grouping

### 5. **Consistent Patterns**
- Same actions work the same way everywhere
- Predictable navigation
- Familiar patterns

### 6. **Gentle & Calm**
- Soft colors
- Smooth animations
- No jarring sounds or alerts
- Reassuring tone

## ðŸ”„ KEY USER FLOWS

### Flow 1: "Who Is This Person?"
1. User sees photo/person
2. Taps "Who is this?" or camera icon
3. App identifies person (or shows options)
4. Shows person card with:
   - Name
   - Relationship
   - Quick facts
   - Recent memories together
5. Can ask: "Tell me more about [Name]"
6. Chatbot provides detailed information

### Flow 2: "What Should I Do Now?"
1. User opens app or asks chatbot
2. Shows today's schedule:
   - Current time highlighted
   - Next 3 items
   - Completed items checked off
3. Can tap any item for details
4. Can mark as complete

### Flow 3: "I Need Help"
1. User taps "Emergency Info" button
2. Shows emergency card:
   - Emergency contacts
   - Medical information
   - Home address
3. Can call contacts directly
4. Can share location

### Flow 4: Adding a Memory
1. Caregiver takes photo
2. Taps "Add Memory"
3. Selects who's in photo
4. Adds date and context
5. Optionally records voice note
6. Saves to person's profile
7. App can later answer "Who is this?" with this memory

### Flow 5: Medication Reminder
1. Time arrives for medication
2. Notification appears (gentle sound)
3. Shows medication card:
   - Name
   - Dosage
   - Photo of pill
   - "I took this" button
4. User taps button
5. Marked as complete
6. Caregiver sees completion
